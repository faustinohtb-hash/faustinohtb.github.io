Solución de problemas - Error de conexión

El error "Load failed" suele estar relacionado con problemas de CORS (Cross-Origin Resource Sharing) o con la configuración del script de Google.

Solución recomendada: Utilizar un proxy CORS para evitar problemas de conexión. Hemos implementado esta solución en el botón "Usar proxy CORS".

Alternativas:

  1. Usar proxy CORS: Esta opción redirige la solicitud a través de un servicio que evita los problemas de CORS.
  2. Configurar el script de Google:
    • Asegúrate de que el script de Google Apps Script esté publicado como "Aplicación web"
    • En la configuración de publicación, selecciona "Ejecutar como: Yo"
    • Asegúrate de que la opción "Quién tiene acceso" esté configurada como "Cualquier persona, incluso anónima"
  3. Ingresar datos manualmente: Si ninguna de las opciones anteriores funciona, puedes ingresar los datos del alumno manualmente.
